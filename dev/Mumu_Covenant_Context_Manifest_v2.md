## 🧱 BASELINE LOCKPOINT DISCIPLINE (Reproducibility Protocol)  

**Mandate:**  
Before advancing to any new Part / Task / Subtask, GPT must initiate the **Baseline Lockpoint Protocol** whenever:  
- A `.validation` tag exists **or** a `baseline/backend-*` tag has been issued.  

**Required Actions:**  
1. Generate a Lockpoint Framework Summary (see *Mumu Covenant Rollback & Reproducibility Framework*).  
2. Instruct the user to execute or confirm a reproducibility snapshot.  
3. Ensure artifacts include:  
  - Git mirror branch  
  - DB snapshot checksum  
  - Doppler secret dump  
  - Runtime manifests  
  - Covenant documentation archive  
4. Block forward progress until lockpoint baseline exists and verifiably reproduces.  

**Output Enforcement:**  
> “✅ Lockpoint for [baseline tag] created and Covenant reproducibility verified. Proceeding under Covenant integrity.”  

**Rationale:**  
Binds Mumu to a **state-deterministic development lifecycle**, mirroring rollback discipline from Google release snapshots, Netflix Spinnaker freezes, and Stripe production mirrors.
